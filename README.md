# Iryna & Gabriel — Guitar Duo Portfolio

A single-page portfolio website for **Iryna & Gabriel**, an acoustic guitar duo based in Vernon, BC, serving the Okanagan and Shuswap regions.

## About the Site

The site is designed to attract event planners, wedding coordinators, and private clients looking for live music. It presents the duo's sound, story, and booking contact in a clean, elegant dark-themed layout built entirely with HTML and CSS — no frameworks or dependencies required beyond a Google Fonts import.

## Sections

- **Hero** — Full-screen intro with duo portrait, tagline, and a call-to-action
- **Live Showcase** — Embedded YouTube performance video
- **The Experience** — Four feature cards covering the duo's offerings: vineyard atmosphere, vocals, guitar, and wireless rig
- **Meet the Duo** — Individual bios for Iryna (Ukraine → Vernon) and Gabriel (Quito → Barcelona → Vernon), each with a photo slot
- **Captured Moments** — Performance photo gallery
- **Contact** — Booking inquiry via email

## Tech

- Pure HTML5 & CSS3 — no JavaScript frameworks
- Google Fonts: Playfair Display + DM Sans
- Fully responsive (mobile, tablet, desktop)
- 100% wireless — single `index.html` file with a local `images/` folder
